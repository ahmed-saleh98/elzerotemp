# Elzero Template By Ahmed Saleh

## Technologies Used

1.  React
2.  Sass
3.  GreenSock
4.  Bootstrap
5.  React-icons

## Installation

run:

```
yarn install
yarn start
```

or

```
npm install
npm start
```
